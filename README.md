SHORT DESCRIPTION ABOUT THE PROJECT:

This project is a contact form built using HTML, CSS, and JavaScript with client-side validation. It allows users to enter their name, email, and message, and validates the inputs before submission. The form checks for empty fields and verifies the email format using regular expressions. Error messages are displayed dynamically below each input if validation fails, and form submission is prevented until all inputs are valid. A success message is shown on successful submission without reloading the page. The user interface was enhanced with modern styling to provide a clean and interactive experience.
